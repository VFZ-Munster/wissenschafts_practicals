
# Configuration for Live server to move between lenguages

docs/
  ├── all folder for en
  ├── de/

# this following liveserver setting is in case of this structure
docs/
  ├── all folder for en
  ├── de/

{
  "liveServer.settings.root": "/docs",
  "liveServer.settings.port": 6324,
  "folders": [
    {
      "path": "."
    }
  ]
}

Then, 
	•	Right-click your docs folder → “Open with Live Server”
	•	Or open any .html file inside docs and start Live Server


# Render in this order othersise the german gets overwritten
quarto render --profile english
quarto render --profile german
