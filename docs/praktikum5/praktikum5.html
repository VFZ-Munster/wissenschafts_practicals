Practical 5: Linear Regression

Outcome Variable

Also known as: response, predicted, dependent variable

  • Type: Continuous
  • 1 variable

Influencing factor(s)

Also known as: covariate, independent, explanatory, predictor variable

  • Type: Continuous or Categorical
  • One (Univariate) or Multiple (Multivariate)

Practicum Scenario

In this practical, we use univariate linear regression to study the association between:

  • Covariate: Smoking (smk)
    • Levels: 1=Never, 2=Former, 3=Current
    • Reference: Never smokers
  • Outcome: Attachment loss (cal_mean)
    • Unit: mm

Exercise 1: Univariable Analysis

Exercise 1.1 Dummy Variables

Build a univariable linear regression model to study the association between smoking (smk; 1=Never, 2=Former, 3=Current, set the never smokers as a reference group) and attachment loss (cal_mean; in mm).

Since the smoking variable has three levels, the first step is to create dummy variables.

1. Create a dummy variable for ‚ÄúFormer‚Äù smokers:

üí° SPSS:

Transform ‚Üí Recode into Different Variables

Step 2: Select smk >> write Former under Name & Label >> press Change.

Step 3: Click ‚ÄúOld and New Values‚Äù.

Step 4: Configure values:

  • Under Old Value: enter 2 >> Under New Value: enter 1 >> Press Add.
  • Under Old Value: enter 1 >> Under New Value: enter 0 >> Press Add.
  • Under Old Value: enter 3 >> Under New Value: enter 0 >> Press Add.
  • Under Old Value: select System- or user-missing >> Under New Value: select System-missing >> Press Add.
  • Select All other values >> select Copy old value(s) >> Press Add. (This handles missing values)

Note: Ensure Measure is Nominal in Variable View, Decimals is set to 0, and Change Values to 0 = ‚ÄúCurrent/none‚Äù and 1 = ‚ÄúFormer‚Äù.

2. Repeat the same procedure to create a dummy variable for ‚ÄúCurrent‚Äù smokers. (Recode smk: 3 -> 1, 1 -> 0, 2 -> 0).

3. Fit the model

üí° SPSS:

Analyze ‚Üí Regression ‚Üí Linear

Step 2: Choose cal_mean as Dependent Variable.

Step 3: Choose Former and Current under Block 1 of 1 (Independent variables).

The linear regression model is defined as:

\[Y_i = \beta_0 + \beta_{former} X_i + \beta_{current} X_i + \epsilon_i\]

where \(\beta_0\) is the intercept term and represents the value corresponding to the reference group or never smokers.

Based on the output, and considering never smokers as the reference group, do individuals who are currently smoking have higher cal_mean measurement than former smokers?: ‚úÖ Interpret the results

(Constant) mean CAL for never smokers 0.942 <0.001 0.913 0.971
Former difference in mean CAL between former smokers and never smokers 0.339 <0.001 0.289 0.388 So former smokers have, on average, 0.34 units higher CAL than never smokers
Current difference in mean CAL between current smokers and never smokers 0.519 <0.001 0.464 0.573 So current smokers have, on average, 0.52 units higher CAL than never smokers

Summarizing: Compared with never smokers, former smokers have about 0.34 higher mean clinical attachment loss, and current smokers have about 0.52 higher mean clinical attachment loss.

Exercise 2: Linear Regression Assumptions

Basically, the results from a linear regression are valid if the assumptions of linear regression are met. These assumptions are:

  • Linearity between the outcome and the covariates
  • Independence of the observations
  • Normality of the residuals
  • Homoscedasticity: constant variance of the residuals

Note: For more details, check the lecture.

Background

By design:

  • We ensure that the observations are derived from independent participants.
  • We ensure that the target variable of our model is normally distributed.

Exercise 3: Effect Modification

We hypothesize that the impact of alcohol on attachment loss differs by gender.

Exercise 3.1 Create an interaction term

Introduce an interaction term between gender (sex) and alcohol consumption (alcg).

1. Create interaction variable (AlcSex):

üí° SPSS:

Transform ‚Üí Compute variable

In the Compute Variable window, in the Target Variable field, type AlcSex.

In the Numeric Expression field, type alcg * sex.

2. Fit multivariable model:

üí° SPSS:

Analyze ‚Üí Regression ‚Üí Linear

Step 2: Dependent: attachment loss

Step 3: Independent: alcg, AlcSex.

Equation: \(Y_i = \beta_0 + \beta_1 alcg + \beta_2 Sex + \beta_3 alcg*Sex + \epsilon_i\)

Interpretation:

  • Intercept: The expected attachment loss of an adult Female with 0g alcohol.

  • alcg: Each 1-unit increase in alcohol consumption is associated with a 0.002 increase in attachment loss.

  • Sex: Males have, on average, a 0.336 increase in attachment loss compared to females.

  • alcg:Sex: Whether alcohol consumption has an impact on attachment loss depends on gender.
